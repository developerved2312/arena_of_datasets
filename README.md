# 📊 Event Dataset Repository  

A curated collection of datasets from various events, competitions, hackathons, and technical challenges.  

This repository is designed to help students, developers, and researchers access structured datasets for learning, experimentation, and competitive problem solving.  

---

## 🚀 Purpose  

- Provide ready-to-use datasets for coding competitions  
- Support hackathons and technical events  
- Encourage data-driven innovation  
- Promote structured problem-solving practice  

---

## 📂 What You'll Find  

- Event-wise dataset folders  
- Problem statements (where applicable)  
- Sample input/output files  
- Cleaned and structured data formats
